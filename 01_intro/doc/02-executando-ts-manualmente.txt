01) Compilando TS para JS:

    Código TS:
    const a: string = 'Teste TS!';
    console.log(a);

02) Comando para compilar:

    OBS: Devemos nos certificar de estarmos dentro da pasta do arquivo 
    TS que será compilado. 

    tsc nomedoarquivo.ts

03) Executando script transpilado manualmente:

    node nomedoarquivotranspilado.js
