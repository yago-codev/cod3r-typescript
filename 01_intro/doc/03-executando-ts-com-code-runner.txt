01) Code Runner é um plugin do VSCode e deverá estar instalado. 

02) Ao tentarmos executar um arquivo TS, 
    receberemos um erro. 

    Para corrigirmos o erro, 
    deveremos instalar uma lib chamada 
    ts-node

    Instalando ts-node globalmente:
    npm i -g ts-node

03) Atalho para executar arquivo TS com o Code Runner:

    control + alt + n
