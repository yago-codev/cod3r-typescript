01) Inferencia de tipos:

    // string
    let nome = 'Yago';
    console.log(nome);
    nome = 28;

    Teremos um erro ao tentarmos reatribuir um valor do tipo string, 
    porque havíamos inicializado o let nome com um valor de string. 

    Isso se chama 
    inferencia de tipos. 

    No momento em que inicializamos uma variável,
    essa variável automaticamente é convertida para o respectivo tipo do valor que foi atribuído.
    De acordo com essa tipagem inplicita, o TS fará a inferencia de tipo,
