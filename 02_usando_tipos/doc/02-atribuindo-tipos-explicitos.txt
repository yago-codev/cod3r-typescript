01) Tipos explícitos:

    Quando não inicializamos uma variável com a atribuição de um valor, 
    ou seja, quando a variável é inicializada somente com o nome, 
    a inferencia de tipos não será feita.

    Ou seja:

    let minhaIdade;
    minhaIdade = 27;
    console.log(typeof minhaIdade); // number
    minhaIdade = 'idade é 27';
    console.log(typeof minhaIdade); // boolean

02) Utilizando o "any" para 
    atribuir uma tipagem dinâmica na variável declarada com TS:

    let minhaIdade: any;
    minhaIdade = 27;
    console.log(typeof minhaIdade);
    minhaIdade = 'idade é 27';
    console.log(typeof minhaIdade);

    Além do tipo "any", podemos usar:
    number, string...

03) Refatorando as variáveis para tipos explicitos:

    let nome: string = 'Yago'; <==
    console.log(nome);
    // nome = 28;


    // number
    let idade: number = 27; <==
    // idade = 'Ana';
    idade = 27.5;
    console.log(idade);


    // boolean 
    let possuiHobbies: boolean = false; <==
    // possuiHobbies = 1;
    console.log(possuiHobbies);


    // tipos explícitos
    let minhaIdade: number; <==
    minhaIdade = 27;
    console.log(typeof minhaIdade);

04) Resumindo:

    Sempre quando não declararmos o tipo explícito de uma variável, 
    o TS automaticamente atribuirá o tipo "any".
